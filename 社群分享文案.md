做了一个小东西。开源了。

---

把一个人的公众号文章、直播稿、知识星球帖丢进去，它会拆成可复用的材料，提取这个人的判断方式，然后批量生成判断力接近他的新文章和短视频文案。

不是仿写。不是学口癖。是提取判断方式。

树林是第一个样本。跑了 8 篇，其中 1 篇 5 转发。果粒看了说方向对。

---

GitHub：https://github.com/hou-152/content-engine

里面有三样东西：
1. 一个 skill（加载到 Claude Code / Codex 就能跑）
2. 一份产品文档（怎么用）
3. 一份详细 README（架构 + 验证数据）
4. 一份心路历程（Phase 0-6，从 884 篇文章到一个引擎）
