# 决策树：树林题材路由

日期：2026-06-16
来源：反赚钱人格（4284字）/ 拒绝（1336字）/ 表达 / 聪明人陷阱 等原文 + 5篇生成稿数据
原则：不是「树林用了什么手法」。是「面对什么选择→为什么这样选」。

---

## 题材分类

| 题材 | 定义 | 代表文章 |
|------|------|---------|
| 对抗-逼认型 | 拆穿读者的自欺，让他承认自己在逃避某事 | 反赚钱人格、免费是最贵的定价、聪明 |
| 对抗-揭露型 | 拆穿一个社会现象，指出读者被收割 | 注意力战争（待验证） |
| 自省-接纳型 | 让读者接受自己的某个状态，不逼他改 | 拒绝、表达和表达、不敢收礼物 |
| 自省-转变型 | 指出一个改变方向，给具体动作 | 拒绝（含 SOL 段） |
| 算账-震慑型 | 用具体数据和计算制造焦虑/紧迫感 | 这是一篇只会让你焦虑的文章 |

---

## 决策点 1：开头策略

```
IF 题材=对抗-逼认型
  → 开头=硬砸判断（一句结论，不铺垫）
  → 原因：读者防御高，温和开头等于给他逃跑时间
  → 示例：「你赚不到钱的根本原因是你讨厌赚钱。」

IF 题材=对抗-揭露型
  → 开头=重复三次（同一判断三种说法）
  → 原因：揭露型需要先降低读者的「事不关己」感
  → 示例：「这个思维不除，你很难不穷。这个思维不除，你有钱了也容易返贫。这个思维不除…」

IF 题材=自省-接纳型
  → 开头=场景代入（让读者先看到自己）
  → 原因：接纳型读者已经知道自己有问题，不需要砸判断
  → 示例：「有人夸你的时候，你是不是第一反应是解释。」

IF 题材=算账-震慑型
  → 开头=「给我一点时间，你看着我跟你算一笔账。」
  → 原因：算账型需要读者承诺注意力，开头就得要。
```

## 决策点 2：CAS 密度与形式

```
IF 读者防御高（对抗-逼认型）
  → CAS 形式=子弹式「你是不是…」清单（4-5条）
  → 密度=高。在砸判断前先铺 4-5 个场景让读者自己对号入座

IF 读者防御中（对抗-揭露型 / 自省-转变型）
  → CAS 形式=单个展开场景（1-2个）
  → 密度=中。每个场景展开 3-5 行

IF 读者防御低（自省-接纳型）
  → CAS 形式=读者内心独白引用
  → 密度=低。1-2 处，每处 2-3 行
```

## 决策点 3：意象使用

```
IF 判断高度抽象（「聪明是一种陷阱」）
  → 需要意象。用具体物体绑住抽象判断。
  → 意象从当前场景中自然生长（「后路」「笼子」）

IF 判断足够具体（「你不敢标价是因为怕被说不值」）
  → 可跳过。判断本身已有物理感。
  → 但需要 S3 场景补足画面感。

IF 判断是情绪型（「你不相信自己值得被爱」）
  → 意象用身体部位（「半合的手」）
  → 身体 > 物体。情绪需要身体来承载。
```

## 决策点 4：收束策略

```
IF 题材=对抗-逼认型
  → 极硬收束（四字以内）
  → 「标价。发出去。」「别准备了。出手。」

IF 题材=对抗-揭露型
  → 反思式收束
  → 「我还是更喜欢那群心里面放着梯子的愣头青。」

IF 题材=自省-接纳型
  → 接纳式收束
  → 「先接受现状，再慢慢调整。」「试试就知道了。」

IF 题材=自省-转变型
  → 指令式收束（中等硬度）
  → 「你犹疑不定的时候，就拒绝。」

IF 题材=算账-震慑型
  → 不回撤。算账的压力不释放。
  → 震撼留在那里，让读者自己消化。
```

## 决策点 5：元反思触发条件

```
IF 前文判断堆积 ≥3 层（OPI→CAS→OPI→CAS→OPI）
  → 需要元反思。读者被压太久了。
  → 位置：S7 硬收束之前。

IF 前文判断 ≤2 层
  → 不需要。元反思会让短文章泄力。

IF 题材=自省-接纳型
  → 不需要。接纳本身就是元反思。

IF 题材=算账-震慑型
  → 不需要。算账的压力需要持续，不能中途给呼吸。
```

## 使用方式

```
生成前：
  1. 判断题材 → 查决策点 1（开头策略）
  2. 估计读者防御 → 查决策点 2（CAS密度）
  3. 生成到 S5 位置 → 查决策点 3（是否需要意象）
  4. 生成到 S6 位置 → 查决策点 5（是否需要元反思）
  5. 生成到 S7 位置 → 查决策点 4（收束策略）

注意：
  决策点是路由，不是模板。条件变了，选择就变。
  如果一篇稿子横跨两个题材，手动选择。
  如果决策点没有覆盖当前情况 → 写完再判断，记入 Phase 6。
```
